# Users CRUD

API para consulta de informações sobre uma empresa com base no CNPJ informado.

## Criada utilizando

- Typescript;
- Axios;
- Ant Design;
- [Users REST API](https://github.com/Fridry/BL-Users-Backend)

## Instalação

    yarn install

## Iniciar a Aplicação

    yarn start

A aplicação irá iniciar em: http://localhost:3000
